egg.extension. dom

A Web page is a document. This document can be either displayed in the browser window, or as the HTML source. But it is the same document in both cases. The Document Object Model (DOM) provides another way to represent, store and manipulate that same document.

<static> listen(element, type, listener) → {void}

Method registers the specified listener on the EventTarget it's called on. The event target may be an Element in a document, the Document itself, a Window, or any other object that supports events.

Name Type Description
element HTMLElement Target to attach listener to
